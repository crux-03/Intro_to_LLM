Attention Mechanism ML & LLM Concepts
A mechanism that computes a weighted combination of value vectors using similarity scores between query and key vectors, allowing every position to directly access information from every other position.
def attention(Q, K, V):
    d_k = K.shape[-1]
    scores = Q @ K.transpose(-2, -1) / math.sqrt(d_k)
    weights = F.softmax(scores, dim=-1)
    return weights @ V

Autoregressive Generation ML & LLM Concepts
Producing text one token at a time by feeding each predicted token back as input for the next step. The loop runs until <EOS> is generated or a maximum length is reached.
for _ in range(max_new_tokens):
    logits, _ = model(input_ids[:, -max_seq_len:])
    next_logits = logits[:, -1, :]        # last position
    probs = F.softmax(next_logits, dim=-1)
    next_token = torch.multinomial(probs, 1)
    input_ids = torch.cat([input_ids, next_token], dim=1)

Byte Pair Encoding (BPE) ML & LLM Concepts
A subword tokenization algorithm that iteratively merges the most frequent adjacent token pair until a target vocabulary size is reached. Used by GPT-2/3/4 and implemented from scratch in Week 2.
def count_pairs(vocab):
    pairs = Counter()
    for word, freq in vocab.items():
        tokens = word.split()
        for i in range(len(tokens) - 1):
            pairs[(tokens[i], tokens[i+1])] += freq
    return pairs

def collate_fn(batch, pad_id=0):
    max_len = max(len(b['input_ids']) for b in batch)
    for b in batch:
        pad = max_len - len(b['input_ids'])
        b['input_ids'] = torch.cat([b['input_ids'], torch.full((pad,), pad_id)])

Feed-Forward Network (FFN / MLP) ML & LLM Concepts
A two-layer MLP with GELU activation inside every transformer block. Expands the hidden dimension 4× (d_model → 4×d_model → d_model) then projects back, adding non-linear capacity.
class FeedForward(nn.Module):
    def __init__(self, d_model, d_ff=None):
        super().__init__()
        d_ff = d_ff or 4 * d_model
        self.fc1 = nn.Linear(d_model, d_ff)
        self.fc2 = nn.Linear(d_ff, d_model)

    def forward(self, x):
        return self.fc2(F.gelu(self.fc1(x)))

Gradient Descent ML & LLM Concepts
An iterative optimisation algorithm that updates parameters by subtracting a fraction (the learning rate) of the gradient: x = x - lr * grad. The course demonstrates it converging to the minimum of a parabola before applying it inside training loops.
lr = 0.1
for step in range(20):
    loss = x ** 2          # f(x) = x²
    loss.backward()
    with torch.no_grad():
        x -= lr * x.grad    # update: x = x - lr * grad
    x.grad.zero_()

The process of producing new text by repeatedly predicting and appending the next token. The course implements it from scratch with a generate() function using various decoding strategies.
@torch.no_grad()
def generate(model, input_ids, max_new=50, temp=1.0):
    for _ in range(max_new):
        logits, _ = model(input_ids[:, -max_seq:])
        probs = F.softmax(logits[:, -1, :] / temp, dim=-1)
        next_tok = torch.multinomial(probs, 1)
        input_ids = torch.cat([input_ids, next_tok], dim=1)
    return input_ids

Top-k Sampling ML & LLM Concepts
Restricting the next-token selection to the k highest-probability tokens before sampling. Prevents the model from choosing very unlikely tokens while still allowing creative diversity.
def top_k_sample(logits, k=50, temperature=1.0):
    scaled = logits / temperature
    top_k_logits, top_k_idx = torch.topk(scaled, k)
    probs = F.softmax(top_k_logits, dim=-1)
    idx = torch.multinomial(probs, 1)
    return top_k_idx.gather(-1, idx)

Top-p (Nucleus) Sampling ML & LLM Concepts
Dynamically selecting the smallest set of tokens whose cumulative probability ≥ p, then sampling from that set. Adapts the candidate pool size to the model's confidence at each step.
sorted_probs, sorted_idx = torch.sort(probs, descending=True)
cumsum = torch.cumsum(sorted_probs, dim=-1)
mask = cumsum <= p
mask[..., 0] = True   # always keep the top token
filtered = probs.clone()
filtered[sorted_idx[~mask]] = 0
return torch.multinomial(filtered / filtered.sum(), 1)

Transformer Block ML & LLM Concepts
The repeating structural unit of GPT, containing LayerNorm, multi-head causal attention, a feed-forward network, and residual connections. GPT-2 Small stacks 12 of these blocks.
class TransformerBlock(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, ...):
        super().__init__()
        self.ln1 = nn.LayerNorm(d_model)
        self.ln2 = nn.LayerNorm(d_model)
        self.attn = MultiHeadCausalAttention(...)
        self.ffn = FeedForward(d_model, d_ff)

    def forward(self, x):
        x = x + self.attn(self.ln1(x))   # residual + attention
        x = x + self.ffn(self.ln2(x))    # residual + FFN
        return x
