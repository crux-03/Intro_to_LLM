Part 01

The Chatbot Class

A chatbot is more than just a model. It wraps the model with conversation history, system prompts, and a clean interface.

class Chatbot:
    def __init__(self, model, tokenizer, system_prompt=None):
        self.model = model
        self.tokenizer = tokenizer
        self.system_prompt = system_prompt
        self.history = []

    def chat(self, message, temperature=0.7):
        # Encode prompt with optional system message
        prompt_ids = self.tokenizer.encode_prompt(
            message, self.system_prompt
        )
        # Generate response
        output = self.model.generate(prompt_ids, end_id=END)
        response = self.tokenizer.decode(output)
        # Save to history
        self.history.append({'user': message, 'assistant': response})
        return response

Part 02

System Prompts for Personality

System prompts let you customize the chatbot's behavior without retraining. The system message is prepended to the prompt with a <SYS> token.

# With system prompt:
"<SYS> You are a friendly teacher. <USER> What is the sun? <ASST>"

# Without system prompt:
"<USER> What is the sun? <ASST>"
