▸ Architecture Inspector
Weight tying: lm_head.weight = tok_emb.weight — the same matrix maps words→vectors and vectors→words. This reduces parameters by ~30% and often improves performance.

Part 03

Training Pipeline

Training combines loss masking (only learn from responses) with gradient clipping and the AdamW optimizer.

def train_chatbot(model, dataset, epochs=35):
    loader = DataLoader(dataset, batch_size=8, shuffle=True)
    opt = AdamW(model.parameters(), lr=1e-3)

    for epoch in range(epochs):
        for batch in loader:
            _, loss = model(batch['input_ids'], batch['labels'])
            opt.zero_grad()
            loss.backward()
            clip_grad_norm_(model.parameters(), 1.0)  # stability!
            opt.step()

Part 04

The Chatbot Class

The Chatbot class wraps everything into a clean interface with personality support, conversation history, and temperature control.

class Chatbot:
    def __init__(self, model, tokenizer):
        self.model = model
        self.tokenizer = tokenizer
        self.system_prompt = None
        self.history = []

    def set_personality(self, prompt):
        self.system_prompt = prompt
        self.history = []          # reset on personality change

    def chat(self, message, temperature=0.7):
        prompt_ids = self.tokenizer.encode_prompt(
            message, self.system_prompt
        )
        output = self.model.generate(
            prompt_ids, temp=temperature, end_id=END
        )
        response = self.tokenizer.decode(output)
        self.history.append({'user': message, 'assistant': response})
        return response

Part 05

Testing & Evaluation

The project tests the chatbot on three levels: known queries (from training data), novel variations (unseen phrasing), and personality consistency.

▸ Test Suite
Honest about limits: Our mini model has ~85K parameters and 90 training pairs. GPT-3.5 has 175 billion parameters and millions of training examples. Our chatbot can handle trained patterns and close variations, but will struggle with truly novel queries. That's expected and educational!
