Causal Masking

GPT generates text left-to-right, one token at a time. Causal masking prevents the model from "cheating" by peeking at future tokens — the key constraint that makes autoregressive generation possible.

Part 01

Why Causal Masking?

GPT generates text left-to-right, one token at a time. When predicting "sat" in "The cat sat", GPT should only see "The cat". If it could see "sat" already in the input, it could just copy the answer — learning nothing.

Part 02

The Causal Mask

A causal mask is a lower triangular matrix. Position i can attend to positions 0 through i, but not beyond.

# One line in PyTorch!
mask = torch.tril(torch.ones(seq_length, seq_length))
# [[1, 0, 0, 0],   ← position 0 sees only itself
#  [1, 1, 0, 0],   ← position 1 sees 0, 1
#  [1, 1, 1, 0],   ← position 2 sees 0, 1, 2
#  [1, 1, 1, 1]]   ← position 3 sees all
Part 03

Applying the Mask

The mask is applied before softmax. We set masked positions to −∞ because softmax(−∞) = 0, guaranteeing zero attention to future positions.

▸ Step-by-Step Masking
scores = scores.masked_fill(mask == 0, float('-inf'))
weights = F.softmax(scores, dim=-1)  # -inf → 0
🔢 Why −∞ and not 0? softmax(0) = e⁰ = 1, which is positive — masked positions would still get attention. softmax(−∞) = 0 exactly, truly removing them from the computation.

Part 04

Implementation

class CausalSelfAttention(nn.Module):
    def __init__(self, d_model, max_seq_length=512):
        super().__init__()
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
        # Mask as buffer — saved with model, not learned
        mask = torch.tril(torch.ones(max_seq_length, max_seq_length))
        self.register_buffer('mask', mask)

    def forward(self, x):
        B, T, C = x.shape
        Q, K, V = self.W_q(x), self.W_k(x), self.W_v(x)
        scores = Q @ K.transpose(-2,-1) / math.sqrt(C)
        # Slice mask to actual sequence length
        scores = scores.masked_fill(self.mask[:T,:T]==0, float('-inf'))
        weights = F.softmax(scores, dim=-1)
        return self.W_o(weights @ V)
