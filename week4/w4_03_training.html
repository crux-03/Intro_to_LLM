Part 01

Training Data

We use character-level tokenization — each character is a token. Simple but effective for learning the concepts.

class CharTokenizer:
    def __init__(self, text):
        chars = sorted(list(set(text)))
        self.char_to_idx = {ch: i for i, ch in enumerate(chars)}
        self.idx_to_char = {i: ch for i, ch in enumerate(chars)}
        self.vocab_size = len(chars)

The input/target pairs are created by shifting by one position:

▸ Input/Target Alignment
Every position is a training example! A sequence of length T gives us T predictions to learn from simultaneously. This is extremely efficient.
Part 02

The Training Loop

1
Forward: Get predictions and compute loss
2
Backward: Compute gradients via backpropagation
3
Clip: Prevent exploding gradients (max norm = 1.0)
4
Step: Update weights with AdamW optimizer
5
Repeat: Thousands of times!
for x, y in loader:
    logits, loss = model(x, y)          # Forward
    optimizer.zero_grad()
    loss.backward()                      # Backward
    torch.nn.utils.clip_grad_norm_(      # Clip
        model.parameters(), 1.0)
    optimizer.step()                     # Update
