Part 01

Layer Normalization

Deep networks suffer from unstable activations — values can explode or vanish as they pass through layers. LayerNorm fixes this by normalizing each sample to mean=0, variance=1, then applying learnable scale and shift.

▸ LayerNorm in Action

See how normalization tames wild activations:

# PyTorch makes it simple

# Learnable parameters:
#   weight (gamma): scale factor, shape (d_model,)
#   bias (beta): shift factor, shape (d_model,)

Part 02

Feed-Forward Network (FFN)

Attention mixes information between positions. The FFN processes each position independently, applying a deep non-linear transformation.

1
Expand: d_model → 4×d_model (creates capacity)
2
GELU: Smooth non-linear activation
3
Contract: 4×d_model → d_model (back to original)
class FeedForward(nn.Module):
    def __init__(self, d_model, d_ff=None, dropout=0.1):
        super().__init__()
        if d_ff is None: d_ff = 4 * d_model
        self.fc1 = nn.Linear(d_model, d_ff)   # Expand
        self.fc2 = nn.Linear(d_ff, d_model)   # Contract
        self.dropout = nn.Dropout(dropout)

    def forward(self, x):
        x = F.gelu(self.fc1(x))    # Expand + activate
        x = self.dropout(x)
        x = self.fc2(x)            # Contract back
        return self.dropout(x)
📚 Analogy — Bottleneck Workshop: Imagine a narrow hallway (d_model) opening into a huge workshop (4×d_model). Raw materials enter, get transformed in many parallel workstations, then the finished products funnel back through a narrow exit. The expansion creates room for complex processing.
▸ GELU vs ReLU
GELU is smooth and allows small negative values through. ReLU has a hard cutoff at 0.

Part 03

Residual Connections

The secret to training deep networks: output = x + sublayer(x)

Instead of learning a complete transformation, each layer only needs to learn the residual — a small correction to the input. If a layer has nothing useful to add, it can output near-zero and the input passes through unchanged.

▸ Gradient Flow Comparison
Why it works: Without residuals, gradients must pass through every layer sequentially — they vanish exponentially. With residuals, gradients have a shortcut path that bypasses all layers, ensuring every layer receives a strong learning signal.
🧩 What is the residual connection formula?
A output = sublayer(x)
B output = x + sublayer(x)
C output = x × sublayer(x)
D output = max(x, sublayer(x))
Part 04

The Complete Block

GPT uses Pre-LayerNorm — normalize before each sublayer (not after):

▸ Block Architecture
class TransformerBlock(nn.Module):
    def __init__(self, d_model, num_heads, d_ff=None,
                 max_seq_length=512, dropout=0.1):
        super().__init__()
        self.ln1 = nn.LayerNorm(d_model)
        self.ln2 = nn.LayerNorm(d_model)
        self.attn = MultiHeadCausalAttention(
            d_model, num_heads, max_seq_length, dropout)
        self.ffn = FeedForward(d_model, d_ff, dropout)

    def forward(self, x):
        x = x + self.attn(self.ln1(x))   # Attention + residual
        x = x + self.ffn(self.ln2(x))    # FFN + residual
        return x
Same shape in, same shape out! Input (batch, seq, d_model) → Output (batch, seq, d_model). This means blocks can be stacked arbitrarily deep.
